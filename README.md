# Квестовик — планировщик рейда EFT

Веб-приложение для отслеживания квестов Escape from Tarkov и планирования рейдов: что взять
с собой и что найти — в разрезе локаций. Без регистрации и сервера: данные статические,
прогресс хранится в вашем браузере.

## Страницы

- **Главная** — что это и зачем, контакты, источник данных.
- **Мои квесты** — активные квесты, которые вы взяли. При выполнении квеста следующие по цепочке
  добавляются автоматически. Сверху — быстрая настройка: укажите свой текущий квест, и все
  предыдущие по ветке отметятся выполненными.
- **Все квесты** — полный каталог на русском, с фильтрами и сортировкой. Любой квест можно
  добавить в «Мои квесты». Для каждого квеста видно, что он требует, что открывает и какие квесты
  провалятся при его выполнении. Категории вида «сдать любой предмет из набора» раскрываются в
  список всех подходящих предметов.
- **Сюжетные** — главы историй: Тур, Небеса в огне, Билет и другие. Это большие квесты, подзадачи
  которых открываются постепенно по стадиям. Главу можно начать, а у некоторых — выбрать концовку.
- **План рейда**:
    - **Мой план** — отмеченные квесты, собранные по локациям: что взять с собой / заложить и что
      найти / купить и сдать.
    - **По локации** — карта локации с маркерами целей текущих квестов, выходами и спавнами боссов,
      переключением этажей и полноэкранным режимом. Рядом — что взять с собой, что найти в рейде для
      квестов и из «Списка находок», и сюжетные задачи локации.
- **Список находок** — личные списки предметов по группам с искомым количеством и отметкой
  «найдено». Поиск по полному каталогу предметов.
- **Каппа** — предметы для контейнера Каппа: чек-лист с прогрессом, поиском и скрытием собранного.

У каждой цели квеста есть отметка выполнения — галочка или счётчик «сделано N из M». Это
уменьшает, что нужно взять и сдать в плане, и показывает остаток по сдаваемым предметам. У каждой
цели есть ссылка на вики.

Фильтры квестов: поиск, локация, торговец, действие, статус, фракция и другие. Иконки предметов
открываются в полном размере. Прогресс, план и списки находок хранятся локально в браузере.

## Данные

Квесты, предметы, иконки и карты берутся из открытого проекта
[tarkov.dev](https://tarkov.dev). Сюжетные квесты — с вики
[escapefromtarkov.fandom.com/ru](https://escapefromtarkov.fandom.com/ru).

## Запуск

Откройте `index.html` двойным кликом либо через локальный сервер:

```
python -m http.server 8777
# затем http://127.0.0.1:8777/
```

Сборка для запуска не нужна — это статические файлы.

## Размещение на GitHub Pages

Settings → Pages → Source `Deploy from a branch`, ветка `main`, папка `/ (root)`. Пути
относительные, поэтому сайт работает и в подпапке `username.github.io/<repo>/`.

## Структура

```
index.html  mine.html  all.html  planner.html       — страницы (общий интерфейс рисует app.js)
story.html  finder.html  kappa.html
styles.css            — стили
app.js                — логика и разметка приложения
data/quests.json      — данные квестов (+ .js — фолбэк для file://)
data/items.json       — каталог предметов для «Списка находок» (+ .js)
data/maps-geo.json    — гео-конфиг карт: SVG, координаты, этажи, выходы, боссы (+ .js)
data/story-quests.json— сюжетные квесты, ведутся вручную (+ .js)
build/fetch.mjs       — загрузка квестов с tarkov.dev -> build/tasks_raw.json
build/transform.mjs   — преобразование -> data/quests.json
build/items.mjs       — каталог предметов -> data/items.json
build/maps.mjs        — гео-конфиг карт -> data/maps-geo.json
```

## Регенерация данных

Нужен Node.js 18+.

```
node build/fetch.mjs       # скачать свежие квесты с tarkov.dev
node build/transform.mjs   # пересобрать data/quests.json
node build/items.mjs       # пересобрать каталог data/items.json
node build/maps.mjs        # пересобрать гео-конфиг data/maps-geo.json
```

Схема данных описана в [`data/SCHEMA.md`](data/SCHEMA.md).
